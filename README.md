A simple menu-driven Python unit converter that converts hours, miles, kilograms, degrees, and Celsius values into their equivalent units.# Unit Converter (Python)

This Python program is a simple text-based Unit Converter that allows users to convert different units using a menu-driven system.
The user can choose from multiple conversion options such as hours to minutes, miles to kilometers, kilograms to pounds, degree to radian, and Celsius to Fahrenheit.

Features
Hours → Minutes
Miles → Kilometers
Kilograms → Pounds
Degree → Radian
Celsius → Fahrenheit
Menu-based structure
Runs continuously until the user chooses to exit
